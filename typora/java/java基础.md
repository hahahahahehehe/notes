# 继承

有三种情况下让类无法有继承的子类

1. 存取控制
   1. 类不能被标记为私有，但是类可以不标记公有，非公有的类只能被同一包下的类做出子类
2. 使用final修饰类
   1. 这表示它是继承树的末端，不能被继承
3. 让类只拥有private的构造函数



如果想要父类的方法不被子类覆盖

1. 将方法加上final修饰符
